JavaScript


Le JavaScript est un langage de programation de script. Orienté objet, le JavaScript utilise le principe de prototypage. Un prototype est un objet à partir duquel on peut créer de nouveaux objets.

Le JavaScript fut le premier langage de programation que j'ai utilisé pour apprendre à developper durant ma formation. J'ai appris grace à NODE.js.

J'apprécie le JavaScript par sa simplicité de comprehenssion mais aussi par la puissance du langage qui permet de découvrir la programation fonctionnelle.

Au début, l'aspect ultra modulable ne me plaisait pas car sur des gros projets cela devient vite illisible et difficile à maintenir.
Mais maintenant avec plus de recul et une plus grande rigueur dans ma façon de développer, j'apprécie mieux JavaScript qu'auparavant.

De plus, l'approche fonctionnelle du langage est très intéressante, elle permet de développer de façon plus fiable et avec moins d'erreurs.

Durant mes deux années chez NOTOCORD dans le projet SENSE, j'ai utilisé le JavaScript très souvent. Car on utilise AngularJS pour la partie web de l'application.
C'est le langages de programmation que je maîtrise le mieux après le C# .
