Javascript


Javascript est un langage de programmation orienté objet.

C'est le langage par lequel j'ai débuté ma formation et qui m'a appris les bases de l'informatique et des processus de programmation intrasèques.

Si le corps enseignant a décidé de commencer l'apprentissage avec ce langage, c'est pour la simple raison qu'il est trivial, facile à prendre en main et reste diversifié dans les possibilités qu'il a à offrir.

En plus d'être très utilisé sur les plateformes web, le Javascript fonctionne de façon modulaire ce qui augmente la flexibilité et la compréhension du code source du langage

Après le Python, c'est le langage que je maitrise le mieux.
